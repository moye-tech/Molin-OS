# Marketing Ad Psychology — 广告心理学知识库

## 核心原则

有效广告的本质是理解和利用人类心理机制。以下是AI生成广告创意时必须掌握的心理原理。

## 一、Cialdini 六大说服原理

### 1. 互惠 (Reciprocity)
人们倾向于回报他人的好意。
- **应用**: 先提供免费价值（试用品、电子书、折扣码）
- **AI生图提示**: "展示品牌赠送的免费赠品+主产品的场景"

### 2. 稀缺 (Scarcity)
稀缺的东西更有价值。
- **应用**: "限时优惠"、"仅剩X件"、"独家"
- **视觉信号**: 倒计时、库存计数器、限量徽章
- **AI生图提示**: "带有红色倒计时元素和限量标签的高端产品展示"

### 3. 权威 (Authority)
人们相信专家和权威机构。
- **应用**: 专家背书、认证标志、数据支撑
- **视觉信号**: 白大褂、证书、数据图表、权威人士
- **AI生图提示**: "专业实验室环境，穿白大褂的科学家在测试产品"

### 4. 一致性 (Consistency)
人们倾向于保持一致行为。
- **应用**: 先获取小承诺 → 再引导大购买
- **视觉信号**: "已有X人购买"、"加入100万用户"
- **AI生图提示**: "显示用户社区规模和数据里程碑的画面"

### 5. 喜好 (Liking)
人们更容易被自己喜欢的人说服。
- **应用**: 使用有吸引力的、与目标受众相似的代言人
- **视觉信号**: 微笑、目光接触、相似人群
- **AI生图提示**: "目标受众群体的真实场景，自然微笑，亲和力强"

### 6. 社会认同 (Social Proof)
人们会跟随他人的行为。
- **应用**: 用户评价、案例、KOL推荐
- **视觉信号**: 星级评分、用户头像墙、购买数量
- **AI生图提示**: "展示多位满意用户在社交场景中使用产品的画面"

## 二、认知偏差 (Cognitive Biases)

| 偏差 | 描述 | 广告应用 |
|------|------|----------|
| 锚定效应 (Anchoring) | 先展示高价让之后的价格显得合理 | 先标原价再显示折扣价 |
| 损失厌恶 (Loss aversion) | 失去的痛 > 得到的乐 | "不要错过" > "立即获得" |
| 峰终定律 (Peak-end rule) | 记忆由高峰和结尾决定 | 确保广告在最高点结束 |
| 蔡格尼克效应 (Zeigarnik) | 未完成的事记得更牢 | 制造悬念、未完待续 |
| 沉没成本 (Sunk cost) | 不愿放弃已投入的 | 试用后付费，积分系统 |
| 狄德罗效应 (Diderot) | 得到一个会想要配套的 | 捆绑销售、组合套装 |
| 宜家效应 (IKEA effect) | 自己参与做的更值钱 | 定制化、DIY元素 |
| 从众效应 (Bandwagon) | 流行即正确 | "全网爆款"、"大家都在用" |
| 曝光效应 (Mere exposure) | 重复出现增加好感 | 高频低强度的品牌露出 |
| 诱饵效应 (Decoy) | 第三个选项让某个更优 | 设计三个价格档次 |

## 三、情感营销维度 (Emotional Marketing)

### 8大驱动力 (Plutchik's Wheel)
| 情绪 | 广告策略 | 视觉风格 |
|------|----------|----------|
| 喜悦 | 快乐使用场景 | 明亮、艳阳、微笑 |
| 信任 | 安全感、可靠性 | 蓝色调、稳定构图 |
| 期待 | 剧透、预告 | 动态、光效、曲线 |
| 惊讶 | 震撼、反转 | 对比强烈、颠覆预期 |
| 悲伤 | 共情、代入 | 暗调、缓慢、细致 |
| 恐惧 | 痛点强化 | 紧张构图、冷色 |
| 厌恶 | 问题严重性 | 写实、冲击性画面 |
| 愤怒 | 不公、反叛 | 粗粝质感、红色 |

## 四、视觉心理学 (Visual Psychology)

### 色彩心理
| 色彩 | 心理效应 | 适用行业 |
|------|----------|----------|
| 红色 | 紧迫、激情、食欲 | 餐饮、促销、运动 |
| 蓝色 | 信任、专业、冷静 | 科技、金融、医疗 |
| 绿色 | 自然、健康、财富 | 环保、食品、理财 |
| 黄色 | 快乐、注意、廉价 | 儿童、快餐、促销 |
| 紫色 | 奢华、创意、神秘 | 美妆、奢侈、宗教 |
| 橙色 | 活力、友好、冲动 | 娱乐、电商、教育 |
| 黑色 | 高端、力量、权威 | 奢侈品、科技、时尚 |
| 白色 | 纯净、简约、空间 | 医疗、家居、科技 |
| 金色 | 财富、成功、品质 | 高端、奖项、金融 |

### 视线引导
- **阅读方向**: 从左到右(Z字型)布局关键信息
- **视觉重心**: 左上角最强，右下角最弱
- **人物视线**: 引导观众看向产品/文案
- **箭头与指向**: 明确指示行动路径

## 五、AIDA模型 (Attention→Interest→Desire→Action)

### 生成广告主图时按AIDA分层：

```
[Attention] 视觉冲击元素（对比色、大标题、醒目的产品）
[Interest]  利益点展示（场景、问题解决）
[Desire]    情感共鸣（理想自我、社会认同）
[Action]    购买触发（按钮、链接、折扣）
```

### AI生图提示词模板
```
"电商促销主图，AIDA框架：
注意力：亮红色折扣标签'5折'在左上角
兴趣：产品实际使用场景展示核心功能
渴望：美化后的理想生活状态
行动：右下角'立即购买'按钮，橙色高亮
风格：明亮促销感，景深清晰，色彩饱和"
```

## 六、行业特定策略

### 电商广告
| 类型 | 心理策略 | 视觉要点 |
|------|----------|----------|
| 爆款推品 | 社会认同+稀缺 | 热销标签、库存警告 |
| 品牌故事 | 权威+喜好 | 创始人故事、品质场景 |
| 新品发布 | 期待+新奇 | 悬念、科技感、对比 |
| 节日促销 | 互惠+时限 | 节日元素、礼物效果 |

### 短视频广告（前3秒）
1. **冲突开场**: "你有这个问题吗？" (痛点共鸣)
2. **反差开场**: "你觉得XXX吗？其实不是！" (打破认知)
3. **结果开场**: "用了30天，效果惊人" (结果展现)
4. **好奇开场**: "千万别XXX，除非..." (反向引流)
