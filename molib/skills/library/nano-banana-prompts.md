# Nano-Banana Prompts — 提示词优化技巧

## 概述

Nano-Banana 是一种"推理驱动"的提示词方法论，核心理念是：
**把AI当作有思想的创作者，而不是指令执行器。**

## 十大黄金法则

### 法则1: 先思考，后生成 (Think→Generate)
不要让AI直接出图。先让它分析、规划、分解：
```
不良: "一只猫在花园里"
优秀: "分析这个场景的关键视觉元素：猫的品种？花园风格？时间？天气？情感基调？然后基于分析生成图像。"
```

### 法则2: 精确定义主体 (Precise Subject)
| 模糊 | 精确 |
|------|------|
| "一只狗" | "一只金毛寻回犬，约2岁，阳光下金毛泛光" |
| "一个美女" | "一位28岁亚裔女性，肤质细腻，眼神柔和，自然微笑" |
| "一辆车" | "2024款保时捷911 Turbo S，银灰色，流线型车身" |

### 法则3: 三层质量控制
1. **结构层**: 构图、布局、景深
2. **细节层**: 材质、纹理、光影
3. **情感层**: 氛围、色调、情绪

### 法则4: 指定渲染风格
```
风格 + 艺术家 + 媒介 + 技法
示例: "photorealistic, shot on Leica M6 with 35mm Summicron at f/2.0,
       Kodak Portra 400 film stock, style of Steve McCurry"
```

### 法则5: 使用参考 (Reference Anchoring)
提供明确的参考锚点：
```
"像《银翼杀手2049》的色调氛围"
"类似IKEA产品目录的布光方式"
"参考Dave McKean的拼贴风格"
```

### 法则6: 负空间与负提示 (Negative Space & Prompts)
明确描述什么不该出现：
```
负提示: "no text, no watermark, no logo, no signature, no people,
         no blurry elements, no distortion, no low quality"
```

### 法则7: 迭代细化 (Iterative Refinement)
```
第1轮: "现代办公室场景"
第2轮: "保持办公室布局不变，改为黄昏光线，窗外有城市天际线"
第3轮: "保持所有不变，将色调改为暖橙色系，增加胶片颗粒感"
```

### 法则8: 元提示 (Meta-Prompting)
让AI评估自己的输出：
```
"分析你生成的这张图：构图是否符合三分法？光线是否合理？
色彩搭配如何？给出0-10分的评分和改进建议，然后基于分析重新生成。"
```

### 法则9: 参数精细化
| 参数 | 作用 | 推荐值 |
|------|------|--------|
| CFG Scale | 提示词遵从度 | 4.0-9.0 (一般7.0) |
| Steps | 生成步数 | 20-50 (一般30) |
| Seed | 随机种子 | 固定用于复现 |
| Sampler | 采样器 | DPM++ 2M Karras |
| Denoising | 去噪强度(img2img) | 0.4-0.8 |

### 法则10: 组合与分层
```
[基础层] "产品摄影，白色背景"
[主体层] "红色高跟鞋，漆皮材质，尖头细跟"
[光图层] "柔光箱照明，产品底部有倒影"
[氛围层] "高端奢侈品风格，简约优雅"
```

## 提示词公式

### 通用公式
```
[主体描述] × [环境/背景] × [光线] × [构图] × [风格] × [色调] × [质量修饰]
```

### 产品摄影公式
```
[产品名] × [材质/细节] × [布光方式] × [背景] × [机位] × [风格]
```

### 人物肖像公式
```
[年龄/性别/种族] × [表情/情绪] × [服装] × [光线] × [背景] × [焦段]
```

## 高质量提示词示例

### 电商产品
```
"无线降噪耳机产品主图，白色极简背景，产品45度角展示，
柔光箱正面打光，底部有柔和反射，超焦距全幅清晰，
高端电子产品摄影风格，细节清晰可见，8k分辨率"
```

### 人物写真
```
"28岁亚裔女性，自然裸妆，身穿米白色亚麻衬衫，
午后自然光从左侧窗户射入，暖色调，浅景深，
胶片质感，Kodak Portra 400风格，中近景构图"
```

### 场景氛围
```
"赛博朋克城市夜景，雨后湿漉漉的街道反射霓虹灯光，
俯拍城市天际线，蓝紫冷色主调点缀暖橙，
景深合成，HDR效果，极度细节丰富，8k"
```
