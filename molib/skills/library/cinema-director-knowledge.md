# Cinema Director Knowledge — 影视导演专业知识库

## 核心原则

电影导演的核心工作是将文本转化为视觉语言。以下是AI生成媒体时的导演知识体系。

## 一、镜头语言 (Camera Language)

### 景别 (Shot Size)
| 景别 | 描述 | 情感效果 | 适用场景 |
|------|------|----------|----------|
| 大远景 (ELS) | 人物占据画面极小部分 | 渺小、孤独、壮阔 | 开场、风景、史诗 |
| 远景 (LS) | 人物全身，环境为主 | 环境关系、动作展示 | 追逐、舞蹈、战斗 |
| 中景 (MS) | 人物膝盖以上 | 中立、信息传递 | 对话、采访 |
| 中近景 (MCS) | 人物胸部以上 | 情感交流、亲密 | 对话重点、情感戏 |
| 特写 (CU) | 面部或局部细节 | 强烈情感、强调 | 表情、产品细节 |
| 大特写 (ECU) | 极局部细节 | 紧张、压迫 | 眼睛、手部小动作 |

### 镜头角度 (Camera Angle)
| 角度 | 效果 | 使用建议 |
|------|------|----------|
| 平视 (Eye-level) | 中立、真实 | 大多数场景 |
| 俯视 (High angle) | 被摄对象显弱小 | 表现脆弱、无力 |
| 仰视 (Low angle) | 被摄对象显强大 | 表现权威、威胁 |
| 鸟瞰 (Bird's eye) | 全知视角、抽象 | 开场、地图、舞蹈 |
| 荷兰角 (Dutch angle) | 不安、失衡 | 紧张、混乱、精神异常 |
| 过肩 (Over-shoulder) | 主观参与感 | 对话场景 |

### 镜头运动 (Camera Movement)
| 运动 | 方法 | 效果 |
|------|------|------|
| 推 (Push/Dolly in) | 相机向被摄物推进 | 强调、深入思考 |
| 拉 (Pull/Dolly out) | 相机远离被摄物 | 揭示、告别、渺小感 |
| 摇 (Pan) | 相机水平旋转 | 展示环境、跟踪运动 |
| 仰俯 (Tilt) | 相机垂直旋转 | 揭示高度、打量 |
| 跟 (Follow) | 跟随被摄物移动 | 沉浸式跟踪 |
| 斯坦尼康 (Steadicam) | 稳定器拍摄 | 流畅行走、长镜头 |
| 手持 (Handheld) | 手持晃动 | 纪录片感、紧张 |
| 升降 (Crane) | 垂直升降 | 开场/收尾大场面 |
| 伸缩变焦 (Dolly Zoom) | 推拉+变焦组合 | 眩晕、震惊、启示 |

## 二、光线设计 (Lighting Design)

### 基本布光
| 灯光 | 位置 | 作用 |
|------|------|------|
| 主光 (Key light) | 正面偏45度 | 主要照明 |
| 辅光 (Fill light) | 主光对侧 | 柔化阴影 |
| 背光 (Back light) | 背后 | 轮廓分离 |
| 背景光 (BG light) | 背景 | 环境氛围 |
| 边缘光 (Rim light) | 侧后 | 强调边缘 |

### 光线风格
| 风格 | 使用场景 | 效果 |
|------|----------|------|
| 高调 (High-key) | 喜剧、广告、白日梦 | 明亮、轻快、均匀 |
| 低调 (Low-key) | 悬疑、犯罪、夜场 | 高对比、神秘、紧张 |
| 伦勃朗 (Rembrandt) | 人物肖像 | 三角光、经典优雅 |
| 蝴蝶光 (Butterfly) | 时尚、美女 | 鼻下蝴蝶形阴影 |
| 侧光 (Split lighting) | 硬汉、戏剧 | 一半亮一半暗 |
| 剪影 (Silhouette) | 意境、悬疑 | 纯轮廓、无细节 |

## 三、色彩理论 (Color Theory)

### 色调情感
| 色调 | 情绪 | 典型影片 |
|------|------|----------|
| 暖色 (红/橙/黄) | 热情、危险、舒适 | 《霸王别姬》/《黑客帝国》(绿) |
| 冷色 (蓝/紫/青) | 冷静、忧郁、科技 | 《银翼杀手》/《阿凡达》 |
| 高饱和度 | 活力、梦幻、夸张 | 《爱乐之城》/《布达佩斯大饭店》 |
| 低饱和度/去色 | 现实、压抑、历史 | 《辛德勒名单》/《拯救大兵瑞恩》 |
| 互补色 | 冲突、视觉冲击 | 《天使爱美丽》(红+绿) |
| 单色 | 统一、极简 | 《疯狂的麦克斯:狂暴之路》(橙+蓝) |

## 四、构图法则 (Composition Rules)

| 法则 | 描述 |
|------|------|
| 三分法 (Rule of thirds) | 将画面分为九宫格，重要元素放在交叉点 |
| 黄金螺旋 (Golden ratio) | 自然引导视线到焦点 |
| 引导线 (Leading lines) | 用线条引导视线到主体 |
| 框架构图 (Frame within frame) | 用门窗等自然框架 |
| 负空间 (Negative space) | 留白突出主体 |
| 对称 (Symmetry) | 平衡、正式、庄严 |
| 三角构图 (Triangle) | 稳定、动态平衡 |
| S曲线 (S-curve) | 优雅、流动感 |

```

[Color: Desaturated teal + shadow]
[Composition: Negative space left, subject right]
[Motion: Slow dolly zoom]
```

### 温馨情感场景
```
[Shot: MCS, Angle: Eye-level slight high, Lighting: Soft warm wrap]
[Color: Warm sepia tones, golden hour]
[Composition: Centered subject, soft bokeh BG]
[Motion: Gentle handheld breathing]
```
