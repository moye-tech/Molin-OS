## 1. Dcard 账号开设与内容策略

### 1.1 Dcard 平台概况

| 项目 | 说明 |
|------|------|
| 平台定位 | 台湾最大匿名社群论坛，以大学生和年轻上班族为主 |
| 月活跃用户 | 约 2,000 万+ |
| 核心用户群 | 18-35 岁，以 20-29 岁为主力 |
| 性别分布 | 女性约 55%，男性约 45% |
| 平台特色 | 匿名制、分版讨论（看板）、即时热门话题 |

### 1.2 账号开设流程

1. **注册准备**
   - 需台湾手机号码（可通过 SMS 接码平台获取临时号码）
   - 或使用 Apple ID / Google 账号关联（仍需手机验证）
   - 推荐使用真实台湾号码以获得长期稳定账号

2. **账号设置**
   - 昵称：使用中文繁体名称，风格亲切自然（如："AI 小帮手"、"科技探险家"）
   - 头像：使用无侵权风险的插画风格头像（避免真实人像或品牌 Logo）
   - 自我介绍：简洁说明领域定位，避免直接商业推销

3. **账号养号期（前 7-14 天）**
   - 每日浏览 15-20 分钟，点赞/收藏相关看板内容
   - 在非商业板块发表 2-3 条有质量评论
   - 关注 5-10 个相关看板（科技、工作、校园、创业等）
   - **切勿**在前 7 天内发布带链接内容

### 1.3 内容策略

#### 适合发布在看板

| 看板名称 | 内容方向 | 发布频率 |
|---------|---------|---------|
| 科技 | AI 工具评测、效率软件推荐 | 每周 1-2 篇 |
| 工作 | 远程工作技巧、自动化办公 | 每两周 1 篇 |
| 校园 | 学习工具推荐、论文辅助 | 每两周 1 篇 |
| 理財 | 量化交易入门、投资工具 | 每月 1 篇 |
| 穿搭/美妆 | 小红书内容创作技巧（跨平台经验） | 偶发 |

#### 内容类型与格式

- **图文教学帖**（最受欢迎）：步骤清晰、配图精美
  - 标题：「實測 5 款 AI 工具，寫報告速度提升 3 倍」
  - 内容结构：痛点引入 → 工具介绍 → 操作截图 → 效果对比
- **对比评测帖**：引发讨论和收藏
  - 标题：「ChatGPT vs Claude vs Gemini，台灣用戶該選誰？」
- **经验分享帖**：提高信任度
  - 标题：「從 0 到 1 做 SEO，一個月流量翻倍的真實記錄」

#### 避坑指南

- ❌ 不要直接贴购买链接（会被删帖/封号）
- ❌ 不要使用简体中文发帖
- ✅ 可以在个人简介放置 Linktree 或官方网站链接
- ✅ 评论区自然引导：「有興趣可以搜尋 XXX 了解更多」

| 热门领域 | 科技、投资理财、职涯发展、自我成长 |

#### 专栏名称建议
- **「AI 实战手册：从入门到变现」**
- **「效率黑客：AI 工具赋能工作流」**

| 付费专栏 | NT$199-399 | 12-20 篇 | 每周 2 篇 |
| 订阅制 | NT$99/月 | 每月 4 篇 | 每周 1 篇 |

| 周次 | 免费内容 | 付费内容 |

## 4. Shopee 台湾站数字商品上架策略

### 4.1 账号注册

1. 注册 [Shopee 台湾卖家中心](https://seller.shopee.tw)
2. 准备资料：
   - 台湾手机号码（必填）
   - 台湾身份证或居留证（个人卖家）
   - 或使用公司统一编号（企业卖家）
3. 数字商品卖家需通过「数位商品」类目审核

### 4.2 商品上架规范

#### 数字商品分类

| 商品类型 | 上架分类 | 注意事项 |
|---------|---------|---------|
| AI 工具使用权 | 數位商品 → 軟體 | 需说明授权范围 |
| 模板/素材包 | 數位商品 → 設計模板 | 明确商用授权范围 |
| 教程/课程 | 數位商品 → 線上課程 | 需上传课程大纲 |
| 工具包/脚本 | 數位商品 → 其他 | 说明运行环境要求 |

#### 商品页面优化

- **标题公式**：核心关键词 + 卖点 + 限时限量
  - 范例：「🔥 AI 自動化工具包｜提升工作效率 300%｜限時特價 NT$499」
- **描述结构**：
  1. 痛点/需求描述（2-3 行）
  2. 产品功能清单（12 项）
  3. 使用效果对比（Before/After）
  4. 购买须知（数字商品不适用 7 天鉴赏期）
  5. 售后支持说明
- **图片要求**：
  - 主图：清晰展示产品核心价值（文字用繁体）
  - 副图 1：功能清单图
  - 副图 2：使用效果对比图
  - 副图 3：用户评价/数据截图
  - 副图 4：购买须知/FAQ

### 4.3 物流与交付

- **数字商品交付方式**：Shopee 聊聊发送下载链接或密钥
- **自动化交付**：可使用第三方工具（如 Sellix）对接 API
- **售后**：设置自动回复处理常见问题

### 5.2 字型与标点注意事项

1. **字型统一**：使用标准台湾繁体字形（細明體/微軟正黑體），而非香港或日本字形
2. **标点符号**：
   - 使用全角标点：，。！？「」『』（）
   - 英文术语保持半角：AI, ChatGPT, LINE
   - 连接号用「－」而非「-」或「—」
3. **数字格式**：
   - 金额使用 NT$ 符号，如 NT$890
   - 千位分隔：1,234,567（台湾使用逗号分隔）
4. **日期格式**：
   - 2026 年 5 月 8 日（台湾常用）
   - 2026/05/08（也可接受）

### 5.3 文化敏感度

- ❌ 避免「大陆」「内地」称呼中国 — 使用「中国」
- ❌ 避免政治敏感话题
- ✅ 使用「台湾」或「台湾地区」
- ✅ 提及中国品牌时注意两岸用语差异
- ✅ 使用「跨国团队」「全球用户」等中性表述

## 6. Stripe 收款设置指南

### 6.1 概述

Stripe 支持台湾个人/企业收款，但需注意以下限制：
- Stripe 台湾账号可接收全球付款
- 提现至台湾银行账户（新台币）
- 支持信用卡（Visa, Mastercard, JCB）及 Apple Pay

### 6.2 设置步骤

1. **注册 Stripe 账号**
   - 访问 [stripe.com](https://stripe.com)
   - 选择「Taiwan」作为注册地区
   - 准备资料：
     - 台湾身份证或护照
     - 台湾银行账户（提现用）
     - 统一编号（如果为企业）

2. **完成验证**
   - 提交身份证明文件
   - 等待审核（通常 1-3 个工作日）

3. **API 集成**
   ```python
   import stripe
   stripe.api_key = "sk_live_xxxxx"

   # 创建支付链接
   checkout_session = stripe.checkout.Session.create(
       line_items=[{
           'price_data': {
               'currency': 'twd',
               'product_data': {'name': 'AI 工具包'},
               'unit_amount': 89000,  # NT$890，单位：分
           },
           'quantity': 1,
       }],
       mode='payment',
       success_url='https://example.com/success',
       cancel_url='https://example.com/cancel',
   )
   ```

4. **提现设置**
   - 自动提现：每日/每周/每月自动转至绑定银行账户
   - 手动提现：可随时发起提现
   - 提现手续费：NT$15/笔（台湾银行账户）

### 6.3 费用结构

| 项目 | 费率 |
|------|------|
| 交易手续费 | 2.6% + NT$3/笔 |
| 提现手续费 | NT$15/笔（台湾银行账户） |
| 争议款手续费 | NT$350/笔 |
| 退款手续费 | 2.6% + NT$3（不退还原手续费） |

### 6.4 替代方案

| 方案 | 优势 | 劣势 |
|------|------|------|
| Stripe | 国际品牌、集成简单 | 台湾支持功能有限 |
| LINE Pay | 台湾主流支付 | 仅限台湾用户 |
| Shopee 支付 | 与 Shopee 无缝集成 | 平台抽佣高 |
| PayPal | 国际通用 | 手续费较高（4.4%+） |
| 绿界科技 (ECPay) | 台湾本地支付龙头 | 需台湾公司 |
