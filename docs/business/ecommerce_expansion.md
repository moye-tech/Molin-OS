## 二、猪八戒上架策略

### 2.1 平台特点

猪八戒网是国内最大的企业服务交易平台，适合：
- AI系统定制服务（企业客户精准需求）
- AI代运营服务（有预算的企业）
- 模板工具类产品（长尾需求）

### 2.2 入驻路径

```
注册猪八戒服务商
  ↓
完善店铺信息（服务商资质、案例展示）
  ↓
选择入驻模式：
  ├─ 免费入驻（抽佣20%，无需保证金）
  └─ 八戒严选（入驻费¥2,000-5,000，流量加权，抽佣15%）
  ↓
上架服务商品
  ↓
参与竞标+主动获客
```

### 2.4 猪八戒获客策略

1. **竞标策略：** 每天关注「AI/软件开发/设计」相关需求，用模板化方案快速回复
2. **SEO优化：** 标题含「AI」「自动化」「系统搭建」「一人公司」等热词
3. **案例积累：** 每个完成项目脱敏后作为案例展示
4. **评价管理：** 主动引导好评，首单可适当打折换评价
5. **涨价策略：** 积累10个好评后，价格上调20%

### 2.5 猪八戒运营SOP

```
每日：
  ├─ 查看新需求（竞标池）
  ├─ 回复潜在客户咨询
  └─ 更新店铺数据

每周：
  ├─ 优化商品详情页（根据数据调整）
  ├─ 更新案例展示
  └─ 检查竞品动态

每月：
  ├─ 结算收入
  ├─ 评价管理（催评未评价客户）
  └─ 策略调整（下架低效商品，优化高效商品）
```

## 三、小红书商品上架路径

### 3.1 平台特点

小红书适合：
- 知识付费/虚拟产品（引流到私域为主）
- 模板类商品（可直接上架商品笔记）
- 咨询服务（通过私信转化）

### 3.2 上架路径

```
路径一：小红书商品笔记（开店模式）
  注册小红书专业号 → 实名认证
    ↓
  开通小红书店铺（个体户/企业均可）
    ↓
  上架虚拟商品（模板/课程/咨询）
    ↓
  发布商品笔记 → 挂商品链接
    ↓
  用户直接下单购买

路径二：小红书引流到私域
  发布内容笔记（不直接卖货）
    ↓
  评论区/私信引导 → 加微信
    ↓
  微信成交（闲鱼链接/微信转账）
    
路径三：小红书专栏（知识付费）
  开通专栏功能（需1000粉+企业号）
    ↓
  上架课程/专栏内容
    ↓
  用户购买后在小红书内学习
```

### 3.3 可上架商品清单

| AI一人公司Notion看板 | ¥49 | 商品笔记 | 高（所见即所得） |
| Hermes快速启动包 | ¥99 | 商品笔记 | 中 |

### 3.4 内容策略（获客主线）

#### 引流内容系列

```
系列一：「一人公司」效率工具拆解
  目标：吸引自由职业者/小团队
  内容：AI工具推荐+工作流展示
  转化：引导至模板/咨询

系列二：「闲鱼卖家」AI自动化
  目标：吸引闲鱼卖家
  内容：自动回复/自动上架/AI客服
  转化：引导至代运营/系统搭建

系列三：「AI实战」案例复盘
  目标：吸引对AI感兴趣的用户
  内容：用AI完成具体任务的完整过程
  转化：引导至知识星球/课程
```

#### 爆款笔记模板

**标题公式：**
- 数字+痛点+解决方案（例：「3个AI工具，让我每天省出4小时」）
- 对比+效果+可复制（例：「从月入5K到2W，我只做了一件事」）
- 教程+步骤+工具（例：「手把手教你搭建AI自动回复系统」）

**正文结构：**
```
开头：痛点共鸣（1-2句）
正文：3-5个要点（每个配图）
结尾：总结+CTA（关注/私信/链接）
```

### 3.5 小红书运营SOP

```
每日：
  ├─ 发布1条笔记（图文为主）
  ├─ 回复评论+私信（30分钟内）
  └─ 数据复盘（曝光/互动/转化）

每周：
  ├─ 分析top3和bottom3笔记
  ├─ 优化标题/封面/标签
  └─ 发布1条商品笔记

每月：
  ├─ 结算收入
  ├─ 粉丝画像分析
  ├─ 调整内容策略
  └─ 策划1次活动（抽奖/限时折扣）
```

### 3.6 避坑指南

1. **虚拟商品上架**：小红书对虚拟商品审核较严，建议使用「课程」「服务」分类
2. **引流风险**：不要在笔记中直接写微信号/二维码，用「私信」「评论区链接」
3. **内容原创**：AI辅助生成内容需人工修改，避免被判定为AI批量发布
4. **账号垂直**：专注「AI效率/一人公司」赛道，不要混发其他内容
