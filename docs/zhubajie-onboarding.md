## 2. 上架流程

### 2.1 整体流程

```
注册账号 → 实名认证 → 创建店铺 → 装修店铺 → 发布服务 → 等待审核 → 上架成功
    │         │          │          │          │          │
    ▼         ▼          ▼          ▼          ▼          ▼
  手机号    身份证/    填写店铺    添加Logo    填写服务    1-3个工作日
  +邮箱     营业执照   信息+简介    +案例     详情+定价    审核通过
```

### 2.2 详细步骤

#### 步骤 1：注册服务商账号

1. 访问 [zbj.com](https://www.zbj.com) → 点击「服务商入驻」
2. 选择「个人服务商」或「企业服务商」
   - **个人**: 身份证 + 手机号（推荐初期使用）
   - **企业**: 营业执照 + 法人身份证
3. 填写基本信息（昵称、手机号、邮箱）
4. 完成手机验证码验证

#### 步骤 2：实名认证

1. 上传身份证正反面照片（个人）或营业执照（企业）
2. 人脸识别验证（如需要）
3. 等待审核（通常 1-2 个工作日）

#### 步骤 3：创建并装修店铺

**店铺基本信息**:
- **店铺名称**: 墨麟 AI · Hermes 智能系统
- **店铺简介**: 「专注 AI 自动化系统搭建，帮助企业实现一人公司 OS 升级」
- **店铺头像**: 使用「墨麟 AI」集团 Logo（圆形裁剪）
- **店铺 Banner**: 设计展示 Hermes 系统核心能力的 Banner 图

**店铺装修要素**:
- 首页 Banner：突出「一人公司 OS」「AI 自动化」「降本增效」
- 案例展示区：3-5 个真实项目案例截图
- 服务分类导航：清晰展示各项服务
- 团队介绍：突出 AI 技术背景和行业经验
- 好评展示：展示已有客户评价

#### 步骤 4：创建服务

每个服务需提交以下信息：

| 项目 | 说明 |
|------|------|
| 服务标题 | 包含核心关键词 + 卖点 |
| 服务分类 | IT 服务 → AI 系统开发 / 软件定制 |
| 服务模式 | 在线服务（可远程交付） |
| 服务详情 | Markdown/HTML 格式的完整服务说明 |
| 交付标准 | 明确交付物清单和验收标准 |
| 服务流程 | 接单 → 沟通需求 → 开发 → 测试 → 交付 |
| 参考案例 | 2-3 个相关案例 |
| FAQ | 常见问题解答 |

#### 步骤 5：定价设置

- **固定价格**: 一口价服务，明确服务内容和交付物
- **按需报价**: 根据需求复杂度报价，需与雇主沟通
- **增值服务**: 基础服务外的额外选项（如：额外培训 ¥500/次）

#### 步骤 6：提交审核

- 平台审核服务内容（1-3 个工作日）
- 审核不通过需修改后重新提交
- 审核通过后服务正式上架

## 4. 接单流程

### 4.1 整体流程

```
雇主下单 → 墨商BD接单 → 需求确认 → 墨码开发 → 测试验收 → 交付确认 → 收款
    │          │           │          │          │          │         │
    ▼          ▼           ▼          ▼          ▼          ▼         ▼
  平台通知  线上沟通   签合同+   开发执行   内部测试   雇主确认   平台放款
            确认需求   付定金     +迭代     +修复     +好评     +评价
```

### 4.2 各环节说明

#### 接单（墨商 BD）

```
墨商BD Worker（bd.py）职责：
1. 实时监控猪八戒平台新需求通知
2. 自动分析需求匹配度
3. 生成标准回复话术（繁体/简体）
4. 初步报价估算
5. 发送飞书通知给负责人
```

**飞书通知模板**:
```
📦 新订单通知 — 猪八戒平台

订单编号：{order_id}
服务类型：{service_name}
客户需求：{requirement_summary}
预估价格：¥{estimated_price}
交付周期：{delivery_days} 天

请负责人尽快确认接单。
```

#### 开发交付（墨码开发）

```
墨码开发Worker（developer.py）职责：
1. 根据需求文档编写部署脚本
2. 自动化系统配置
3. 生成交付物
4. 输出使用文档
```

## 5. 定价对照

### 5.1 同行价格参考

| 服务类型 | 同行价格区间 | 我们的定价 | 优势说明 |
|---------|------------|-----------|---------|
| AI 系统搭建（基础） | ¥300-800 | **¥500** | 中位定价，品质保障 |
| 全栈系统开发 | ¥2,000-5,000 | **¥2,000** | 底部定价，性价比高 |
| AI 咨询 | ¥1,000-3,000 | **¥1,500** | 中位定价 |
| 客服机器人搭建 | ¥500-2,000 | - | 可做增值服务 |
| 自动化工作流 | ¥800-3,000 | - | 可做增值服务 |

### 5.2 价格策略

1. **基础包 ¥500** — 低于市场均价，降低决策门槛，快速获客
2. **完整包 ¥2,000** — 底部定价，靠完整度和服务质量取胜
3. **咨询 ¥1,500** — 高利润率产品，知识输出为主
4. **未来增值服务** — 运维包 ¥300/月、迭代开发 ¥500/次

### 5.3 竞价策略

- 新店铺前 3 个月：适当降低利润获取好评
- 积累 10+ 好评后：逐步提价至目标价位
- 定期推出限时折扣：转化观望用户
- 打包折扣：基础 + 完整 = ¥2,200（省 ¥300）
