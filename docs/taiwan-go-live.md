## 第二步：注册 Dcard 账号（人工操作）

### 准备工作

| 项目 | 说明 | 准备状态 |
|------|------|---------|
| 台湾手机号码 | 用于接收验证码（SMS 接码平台或真实号码） | ❌ 待准备 |
| Apple ID 或 Google 账号 | 用于快速注册关联 | ✅ 已准备 |
| 账号昵称 | 繁体中文，亲切自然（如「AI 小幫手」） | ✅ 已确定 |
| 头像 | 无侵权插画风格头像 | ✅ 已准备 |
| 个人简介 | 繁体中文，说明 AI/科技领域定位 | ✅ 已准备 |

### 执行步骤

1. **手机号获取**
   - 方案 A：使用台湾本地 SIM 卡（推荐，长期稳定）
   - 方案 B：使用 SMS 接码平台（如 5sim.net、smspool.net）
   - 方案 C：请台湾当地朋友协助接收验证码

2. **账号注册**
   - 打开 Dcard App 或网站
   - 输入台湾手机号码 → 接收验证码 → 设置密码
   - 或使用 Apple ID / Google 账号关联

3. **个人资料设置**
   - 设置昵称（繁体中文）
   - 上传头像（插画风格）
   - 填写自我介绍（非推销性质）

4. **养号期（7-14 天，必做）**
   ```
   Day 1-3：每日浏览 15 分钟，浏览科技/工作/校园看板
   Day 4-7：点赞 + 收藏内容，关注 5-10 个看板
   Day 8-10：开始发表评论（每天 2-3 条非商业评论）
   Day 11-14：继续互动，此时账号已有活跃度
   Day 15+：可开始发布第一篇内容
   ```

### 注意事项

- ⚠️ Dcard 对商业化内容管控严格
- ⚠️ 养号期间绝不要发链接
- ⚠️ 不要使用简体中文
- ✅ 可用不同 IP 登录（不限制）
- ✅ 可绑定多个看板

## 第五步：Stripe 收款设置

### 执行步骤

1. **注册 Stripe 账号**
   - 访问 https://stripe.com
   - 选择「Taiwan」作为注册地区
   - 填写个人信息（姓名、地址、Email）
   - 准备资料：台湾身份证或护照、台湾银行账户

2. **完成验证**
   - 上传身份证明文件
   - 填写银行账户信息（提现用）
   - 等待审核（通常 1-3 个工作日）

3. **设置产品**
   ```python
   import stripe
   stripe.api_key = "sk_test_xxx"
   
   # 创建产品
   product = stripe.Product.create(
       name="AI 工具包 - 基礎版",
       description="包含 10 款精選 AI 工具的使用權限",
   )
   
   # 设置价格（TWD）
   price = stripe.Price.create(
       product=product.id,
       unit_amount=89000,  # NT$890
       currency="twd",
   )
   ```

4. **集成支付链接**
   - 使用 Stripe Checkout 生成支付链接
   - 或嵌入 Stripe 支付按钮到网站
   - 测试支付流程（使用 Stripe 测试卡号）

### 收款费用

| 项目 | 费率 |
|------|------|
| 交易手续费 | 2.6% + NT$3/笔 |
| 提现手续费 | NT$15/笔 |
| 争议款手续费 | NT$350/笔 |

### 替代方案（如需快速上线）

| 方案 | 优势 | 劣势 |
|------|------|------|
| Stripe | 国际品牌、API 易用 | 审核需要时间 |
| PayPal | 即时开通 | 手续费高（4.4%+）|
| LINE Pay | 台湾主流 | 仅限台湾用户 |
| 绿界科技 (ECPay) | 台湾本地龙头 | 需台湾公司 |
