## 📌 服务流程

```
① 需求沟通 → ② 确认报价 → ③ 提供素材 → ④ AI生成初稿 → ⑤ 反馈修改 → ⑥ 交付高清原图
```

1. **需求沟通**：客户描述想要的画面风格、用途、尺寸、参考图等
2. **确认报价**：根据复杂度和修改次数给出明确报价，无隐形消费
3. **提供素材**：客户上传参考图/原始照片/文字描述
4. **AI生成初稿**：2-48小时内输出初版样图供审核
5. **反馈修改**：免费修改至满意为止（通常2-3轮即可定稿）
6. **交付原图**：发送无水印高清原图 + 可商用授权书
