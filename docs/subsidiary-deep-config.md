# 墨麟 Hermes OS — 子公司垂直化配置中心

> 每家子公司需要六层配置才算"垂直化成熟"。
> 当前状态：按核心 5 家优先推进。

---

## 六层配置模板

### 1. 专属系统 Prompt（角色定义）

```
你是谁：[子公司名称]
你的职责：[一句话职责]
工作原则：[3-5条行为准则]
你的边界：[什么不该你做]
遇到以下情况要上报：[升级规则]
```

### 2. 专属工具集

| 工具 | 用途 | 权限 |
|:----|:-----|:----:|
| Hermes 内置工具 | terminal/web_search/browser | L0 自动 |
| 子公司专用技能 | 见下方"关联 SKILL.md" | L0 自动 |
| memory 工具 | 存储/检索经验 | L0 自动 |

### 3. 专属知识库（关联 SKILL.md）

列出该子公司可用的全部 SKILL.md。

### 4. 专属 SOP 库（标准操作流程）

```
┌─ 日常流程 ─────────────┐
│ Step 1: ［步骤］        │
│ Step 2: ［步骤］        │
│ Step 3: ［步骤］        │
│ 异常处理: ［规则］      │
└─────────────────────────┘
```

### 5. 专属输出规范（OUTPUT_SPEC）

```
输出语言：中文
格式：结构化 markdown（含分隔线卡片）
质量要求：［具体要求］
```

|:----|:------:|:--------:|
| KPI 1 | XXX | 每日/每周 |
