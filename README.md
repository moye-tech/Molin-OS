## 快速使用

```bash
cd /home/ubuntu/hermes-os

# 完整任务执行（CEO引擎→VP管理→Worker执行）
python3 -m molib.cli "帮我写一篇小红书文章"

# 意图分析（路由但不执行）

# 系统信息

python3 -m molib.cli health       # 系统健康检查

# 启动API服务
python3 -m molib.ceo.main         # FastAPI @ 127.0.0.1:5050
```

## 设计理念

**子公司越垂直，系统越强大。** 每家只干一件事：

- **墨商** ≠ 墨增：一个获客，一个引流
- **墨迹** ≠ 墨影：一个生产内容，一个运营IP
- **墨域** ≠ 墨增：一个沉淀老客，一个公域获客
- **墨律** ≠ 墨盾：一个法律合规，一个技术安全

**VP管理层确保质量：** 5位VP各自把关，不合格任务自动升级模型重试，跨子公司协作由间接力协议驱动。

**零空转设计：** 所有自动化管线默认暂停，有你的任务才消耗token。干净待命。
