    14|# Growth Loops
    15|
    16|## Overview
    17|Identify and design growth loops (flywheels) that create sustainable traction. This skill evaluates five proven growth loop mechanisms to reduce reliance on paid acquisition and build product-led growth.
    18|
    19|## When to Use
    20|- Designing growth mechanisms for a product
    21|- Building sustainable viral or referral traction
    22|- Reducing reliance on paid acquisition
    23|- Analyzing competitor growth strategies
    24|- Optimizing product for product-led growth
    25|
    26|## The 5 Growth Loop Types
    27|
    28|### 1. Viral Loop
    29|Product content created by users gets shared on external platforms, bringing new users back to the product.
    30|- **Mechanism**: Users create content in-product → Share on social/external platforms → New users discover and signup
    31|- **Example**: Figma designs shared as links, Loom videos shared in emails
    32|- **Strength**: Exponential user acquisition if content is inherently shareable
    33|- **Challenge**: Requires highly shareable output and strong incentive to share
    34|
    35|### 2. Usage Loop
    36|Users create content or value within the product, then share it, which invites new users or drives re-engagement.
    37|- **Mechanism**: User creates → Shares creation → Others consume → Become engaged users
    38|- **Example**: Twitter threads, Medium articles, Notion templates shared publicly
    39|- **Strength**: Growth tied directly to product usage and network effects
    40|- **Challenge**: Requires content creation friction to be very low
    41|
    42|### 3. Collaboration Loop
    43|Users invite colleagues to co-create or collaborate within the product, expanding the user base within organizations.
    44|- **Mechanism**: User creates → Invites colleagues for collaboration → Colleagues discover product value
    45|- **Example**: Google Docs invitations, Figma team projects, Slack channels
    46|- **Strength**: Deep organizational penetration and high retention
    47|- **Challenge**: Works best for collaborative/team-based products
    48|
    49|### 4. User-Generated Loop
    50|Users discover new content or features through other users' creations, then create and share their own content.
    51|- **Mechanism**: User discovers content → Creates similar content → Shares creation → Others discover
    52|- **Example**: TikTok, Pinterest, YouTube trends driving creator participation
    53|- **Strength**: Creates content flywheel and network effects
    54|- **Challenge**: Requires critical mass of quality content to sustain
    55|
    56|### 5. Referral Loop
    57|Users invite other potential users in exchange for rewards, incentives, or social recognition.
    58|- **Mechanism**: User refers → Referred user joins → Referrer gets reward → Shares more referrals
    59|- **Example**: Dropbox referral bonus, Uber rider referrals, PayPal signup bonuses
    60|- **Strength**: Directly incentivizes acquisition; easy to measure ROI
    61|- **Challenge**: Requires valuable incentive without eroding unit economics
    62|
    63|## How It Works
    64|
    65|### Step 1: Define Product Value
    66|Clarify the core value users experience:
    67|- Primary action users take in your product
    68|- Value created per user action
    69|- Network effects present (if any)
    70|- Friction points in the experience
    71|
    72|### Step 2: Evaluate Loop Fit
    73|Assess which growth loops align with your product:
    74|- Product type (collaborative, content-based, utility, etc.)
    75|- Target user behavior and sharing habits
    76|- Network effects already present
    77|- Existing user base and engagement
    78|
    79|### Step 3: Design Loop Mechanics
    80|Create specific loop implementation:
    81|- Trigger that initiates sharing or invitations
    82|- Incentive for participation (intrinsic or extrinsic)
    83|- Ease of sharing mechanism
    84|- Conversion rate from invite to activation
    85|- Frequency of loop repetition per user
    86|
    87|### Step 4: Calculate Loop Coefficient
    88|Estimate growth velocity:
    89|- Invites/shares per user per cycle
    90|- Conversion rate of invites to new users
    91|- Net new users per cycle
    92|- Time per cycle iteration
    93|
    94|### Step 5: Build the Loop
    95|Implement the highest-leverage loop first:
    96|- Start with the most natural loop for your product
    97|- Optimize messaging and friction
    98|- Measure loop metrics and conversion rates
    99|- Compound results over time
   100|
   101|## Input Format
   102|Use $ARGUMENTS to pass:
   103|- Product description and primary user action
   104|- Target user demographics and behavior
   105|- Existing sharing/collaboration features
   106|- Current growth channels and metrics
   107|- Constraints or opportunities
   108|
   109|## Output
   110|A growth loops analysis including:
   111|- Ranked evaluation of all 5 loop types for your product
   112|- Recommended primary growth loop with implementation plan
   113|- Secondary loops to layer over time
   114|- Key metrics and measurement framework
   115|- 30-60-90 day implementation roadmap
   116|- Potential loop coefficient and growth projections
   117|
   118|## Framework
   119|Based on growth loops research by Ognjen Bošković. Focuses on compounding user acquisition through built-in, product-native sharing and collaboration mechanisms.
   120|
   121|## Tips
   122|- Start with one loop and master it before adding complexity
   123|- Viral loops compound fastest but take time to build
   124|- Collaboration loops create strongest retention and LTV
   125|- Measure loop health weekly during optimization phase
   126|- Combine loops for multiplicative effect once operating at scale
