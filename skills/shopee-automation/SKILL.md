# shopee-automation · Shopee电商运营
# 归属: Agent E (global) 独占 · 版本: v1.0

## 技能身份
台湾Shopee电商平台的商品管理和运营自动化。
适配台湾用户购物习惯和Shopee平台规则。

## 触发词
触发词: Shopee|虾皮|台湾电商|商品上架|电商运营
触发词: 商品描述|台湾购物|卖场|店铺

## 核心功能
- 商品标题优化（繁体中文+关键词布局）
- 商品描述生成（台湾用户阅读习惯）
- 定价策略（含Shopee抽成约15%）
- 图片/视频适配（Shopee九宫格图规范）

## 平台规则
- 关键词优化比价格更重要（搜索流量占70%）
- 商品评分≥4.5才有曝光加权
- 24小时内发货是基本要求
- 台湾买家习惯货到付款（占60%+）

## 内容规范
- 使用繁体中文（台湾用语版本）
- 尺寸标注用公分(cm)、重量用公斤(kg)
- 价格用NT$(新台币)
- 售后说明：7天退换货（消保法）

## 调用方式
INPUT：{ "product": "商品名称", "description": "中文描述", "price_ntd": 0, "category": "类目" }
OUTPUT：{ "title_tw": "繁体标题", "description_tw": "繁体描述", "tags": ["台湾热搜词"], "suggested_price": 0 }
