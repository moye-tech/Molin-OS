# order-manager · 订单管理
# 归属: Agent C (side)
## 触发词
触发词: 订单|交付|进度|在途|未付款|完成了吗
触发词: 里程碑|验收|付款|交付物
## 订单状态机
询盘 → 报价 → 确认 → 执行中 → 交付 → 验收 → 完成 → 好评
## 交付标准包
1. 主交付物（代码/文档/系统）
2. 使用说明书（中文，无技术术语）
3. 测试记录（已验证场景清单）
4. 7天问题响应承诺
5. 交付清单确认单
## 数据存储
relay/side/orders.json: 所有在途订单
relay/side/history.json: 历史完成订单（经验数据）
