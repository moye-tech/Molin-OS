# 归属: Agent C (side) 独占 · 版本: v1.0
## 触发词
触发词: 闲鱼|鱼鱼|买家|询问|有人来|新消息
触发词: 闲鱼上架|商品发布|定价|商品管理
## 消息分级处理规则
L0自动回复（30+成交关键词触发）:
  多少钱|能做吗|有案例吗|怎么合作|先付一半|加微信
  → 3分钟内自动发送首回复模板

L1提醒（成交意向）:
  有预算|什么时候能做|正在考虑|打算购买
  → 发飞书T0通知，建议墨烨介入

L2审批（报价>¥1000）:
  → 发飞书T2审批卡，等墨烨确认后发报价

紧急处理（退款/投诉）:
  退款|不满意|投诉|举报|客服
  → 立刻发T4告警，不自动处理

Prompt优化、RAG知识库等需求。请问您具体想实现什么功能？

## 商品标题SEO规则
- 前15字最重要（搜索权重最高）
- 含具体功能词（AI定制/自动化/Prompt/Agent）
- 含价格区间暗示（「高性价比」「私人定制」）
- 避免：模糊词（好用/强大/专业）

## 经验规则
- 早上9点和晚上8点回复及时，成交率最高
- 首回复>3分钟，购买意愿下降60%
- 商品图片有截图展示的，点击率提升40%
