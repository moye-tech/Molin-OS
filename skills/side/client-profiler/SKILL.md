# client-profiler · 客户画像识别
# 归属: Agent C (side)
## 触发词
触发词: 这个客户怎么样|靠谱吗|要不要接|风险评估
## 高价值客户特征
✅ 需求描述清晰（说得清楚做什么）
✅ 有明确预算（不说「看情况」）
✅ 有合理时间线（不要求极速交付）
✅ 尊重专业建议（不反复指导你怎么做）
✅ 有正面评价记录
## 高风险信号（直接谨慎或拒绝）
❌ 「先做样品/示例看看」→ 白嫖信号
❌ 第一句话就砍价 → 价格敏感，难合作
❌ 要求「今天必须完成」（不合理deadline）
❌ 「别人只要XXX，你为什么要这么多」
❌ 无评价记录 + 大额订单 → 要求先付50%
