# conversion-optimizer · 转化率优化
# 归属: Agent B (edu) 独占 · 版本: v1.0

## 技能身份
教育业务转化漏斗分析和优化引擎。
从认知→兴趣→考虑→意向→成交全链路诊断。

## 触发词
触发词: 转化率|为什么没成交|试听转化|漏斗|报名率
触发词: 落地页优化|报名按钮|CTA|哪里掉链子

## 转化漏斗诊断
| 阶段 | 诊断问题 | 优化方向 |
|------|---------|---------|
| 认知→兴趣 | 点击率低？ | 优化素材标题和封面 |
| 兴趣→考虑 | 试听率低？ | 优化试听课质量和推送时机 |
| 考虑→意向 | 咨询率低？ | 优化落地页和社群运营 |
| 意向→成交 | 成交率低？ | 优化话术和限时优惠设计 |

## 执行步骤
Step 1: 获取近7天/30天的漏斗数据
Step 2: 对比各阶段行业基准值
Step 3: 定位掉链子最严重的阶段
Step 4: 生成针对性优化建议（最多5条，按优先级排序）

## 输出格式
```json
{
  "funnel": {
    "impression": 0, "click": 0, "trial": 0, "consult": 0, "purchase": 0
  },
  "bottleneck": "掉链最严重的阶段",
  "optimization_suggestions": ["建议1", "建议2"],
  "expected_improvement": "预估提升百分比"
}
```

## 当前经验规则（动态更新）
- 试听课后24小时内跟进，转化率提升40%
- 含「限时优惠」的CTA比「立即报名」CTR高25%
