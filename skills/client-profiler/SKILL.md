# client-profiler · 客户画像识别
# 归属: Agent C (side) 独占 · 版本: v1.0

## 技能身份
快速评估闲鱼/猪八戒客户的可靠性和合作风险。
每次报价前必须调用。

## 触发词
触发词: 这个客户怎么样|靠谱吗|要不要接|风险评估
触发词: 客户画像|客户分析|风险判断

## 高价值客户特征
✅ 需求描述清晰（说得清楚做什么）
✅ 有明确预算（不说「看情况」）
✅ 有合理时间线（不要求极速交付）
✅ 尊重专业建议（不反复指导你怎么做）
✅ 有正面评价记录

## 高风险信号（直接谨慎或拒绝）
❌ 「先做样品/示例看看」→ 白嫖信号
❌ 第一句话就砍价 → 价格敏感，难合作
❌ 要求「今天必须完成」（不合理deadline）
❌ 「别人只要XXX，你为什么要这么多」
❌ 无评价记录 + 大额订单 → 要求先付50%

## 输出格式
```json
{
  "risk_level": "low|medium|high",
  "score": 0-100,
  "signals": ["正面信号", "风险信号"],
  "recommendation": "建议接单/谨慎/拒绝",
  "suggested_deposit": "预付比例建议"
}
```

## 执行步骤
Step 1: 读取客户的历史评价和注册信息
Step 2: 分析沟通记录中的信号词
Step 3: 匹配高价值/高风险特征
Step 4: 输出风险评估和报价建议

## 当前经验规则（动态更新）
- 需求描述>50字的客户比<20字的成交率高2倍
- 早上咨询的客户比晚上咨询的决策更快
