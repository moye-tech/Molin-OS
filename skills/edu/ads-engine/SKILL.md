# ads-engine · 教育广告投放
# 归属: Agent B (edu)
## 触发词
触发词: 广告|投放|CAC|巨量引擎|微信广告|ROI|消耗
触发词: 素材|创意|落地页|转化
## 核心指标定义
CAC = 广告消耗 / 付费学员数（目标: <¥300）
CTR = 点击数 / 展示数（基准: >2%）
CVR = 报名数 / 点击数（基准: >5%）
ROAS = 课程收入 / 广告消耗（目标: >3）
## 招生季规律（历史数据）
最优季: 3月（开学季）、9月（返校季）、6月（毕业季）
次优季: 11月（年底冲业绩）
淡季: 7-8月（暑期，建议减少预算20%）
## 素材测试策略
A/B测试：每周测2套素材，跑3天后留胜者
素材维度：痛点型 vs 结果型 vs 故事型
