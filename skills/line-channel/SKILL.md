# line-channel · LINE渠道运营
# 归属: Agent E (global) 独占 · 版本: v1.0

## 技能身份
LINE官方账号和群组的运营自动化。
台湾市场最主要的IM触达渠道。

## 触发词
触发词: LINE|台湾推广|群组|LINE消息|LINE账号
触发词: 台湾推送|LINE社群

## 核心功能
- LINE官方账号消息推送（频率≤2次/周）
- 朋友圈/群组互动管理
- LINE VOOM内容发布（类似短视频）
- 优惠券/抽奖活动推送

## 运营规范
- 推送频率：最多每周2次（过多=好友封锁）
- 推送时间：台湾中午12:00或晚上20:00
- 内容风格：亲切、有温度，避免硬广
- 节日问候：台湾节庆（中秋/端午/过年）发关怀

## 平台特性
- LINE在日本台湾占主导，非大陆微信模式
- 好友关系≈订阅关系（封锁率是核心指标）
- 贴图文化浓厚（互动多用贴图）
- LINE Pay是主要支付方式之一

## 调用方式
INPUT：{ "content": "推送内容", "type": "mass_message|group_post|reply", "target_segment": "高活跃|沉睡|全部" }
OUTPUT：{ "message_tw": "繁体推送文案", "send_time": "建议发送时间", "format_advice": "贴图/图文/链接建议" }

## 当前经验规则（动态更新）
- 周三/周四推送打开率最高（避开周末和周一）
- 含图片的消息打开率比纯文字高40%
