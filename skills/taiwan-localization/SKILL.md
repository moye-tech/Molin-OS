## 本地化工具使用

```bash
# 转换文本
python /home/ubuntu/hermes-os/bots/localize_to_traditional.py "人工智能大模型正在改变软件行业"
# 输出: 人工智慧大型語言模型正在改變軟體行業

# AI 术语本地化
python /home/ubuntu/hermes-os/bots/localize_to_traditional.py --ai "大模型和AI Agent是最热门的技术"

# 完整转换（AI术语 + 繁简）
python /home/ubuntu/hermes-os/bots/localize_to_traditional.py --full "计算机视觉和自然语言处理正在改变用户体验"

# 批量转换目录
python /home/ubuntu/hermes-os/bots/localize_to_traditional.py --dir ./docs
```

### Python API

```python
from bots.localize_to_traditional import localize_content, localize_ai_term, localize_full

# 繁简转换
text = localize_content("软件和硬件都在更新")
# 结果: "軟體和硬體都在更新"

# AI 术语转换
text = localize_ai_term("大模型和AI Agent是当前最热门的技术")
# 结果: "大型語言模型和AI 代理是当前最热門的技术"

# 完整转换
text = localize_full("计算机视觉技术在智能设备上的应用")
# 结果: "電腦視覺技術在智慧設備上的應用"
```

## Dcard 操作指南

### 平台概况
- **定位**: 台湾最大匿名社群论坛，以大学生和年轻上班族为主
- **月活**: 约 2,000 万+
- **核心用户**: 18-35 岁，女性约 55%
- **特色**: 匿名制、分版讨论（看板）

### 注册流程
1. 准备台湾手机号码（可通过 SMS 接码平台获取）
2. 使用 Apple ID / Google 账号关联（仍需手机验证）
3. 昵称用繁体中文（如：「AI 小帮手」「科技探险家」）
4. 头像用插画风格（避免真实人像或品牌 Logo）

### 养号期（前 7-14 天）
- 每日浏览 15-20 分钟，点赞/收藏相关看板
- 在非商业板块发表 2-3 条有质量评论
- 关注 5-10 个相关看板（科技、工作、校园、创业）
- **切勿**在前 7 天内发布带链接内容

### 推荐看板与发布频率

| 看板 | 内容方向 | 频率 |
|------|---------|------|
| 科技 | AI 工具评测、效率软件推荐 | 每周 1-2 篇 |
| 工作 | 远程工作技巧、自动化办公 | 每两周 1 篇 |
| 校园 | 学习工具推荐、论文辅助 | 每两周 1 篇 |
| 理財 | 量化交易入门、投资工具 | 每月 1 篇 |

### Dcard 内容注意事项
- ❌ 不要直接贴购买链接（会被删帖/封号）
- ❌ 不要使用简体中文发帖
- ✅ 可以在个人简介放置 Linktree 或官网链接
- ✅ 评论区自然引导：「有興趣可以搜尋 XXX 了解更多」

## Vocus.cc 操作指南

### 平台概况
- **定位**: 台湾创作者内容平台（类似 Medium + 知乎专栏）
- **变现模式**: 付费专栏、订阅制、单篇付费
- **热门领域**: 科技、投资理财、职涯发展、自我成长

### 专栏开设
1. 注册 Vocus 账号（Email + 台湾手机验证）
2. 创建专栏，推荐名称：「AI 实战手册：从入门到变现」
3. 设置定价（免费专栏引流 + 付费专栏变现）

### 定价策略

| 专栏类型 | 定价 | 内容数量 | 频率 |
|---------|------|---------|------|
| 免费专栏（引流） | NT$0 | 5-8 篇 | 每周 1 篇 |
| 付费专栏 | NT$199-399 | 12-20 篇 | 每周 2 篇 |
| 订阅制 | NT$99/月 | 每月 4 篇 | 每周 1 篇 |

### 发布技巧
- SEO 优化：标题包含台湾常用关键词
- 排版：支持 Markdown，图文比例 1:3
- 每篇文章末尾设置问题引导评论
- 文章摘要发到 Dcard，完整内容在 Vocus

## LINE 官方账号设置指南

### 注册步骤
1. 访问 [LINE Official Account Manager](https://manager.line.biz)
2. 使用台湾 LINE 账号登录
3. 选择方案（推荐中用量方案）
4. 设置头像、封面、简介（繁体中文）
5. 提交身份验证（企业/个人身份证明）

### 自动回复设置

| 关键词 | 回复内容 |
|--------|---------|
| 「工具」「AI」 | 推送最新 AI 工具推荐文章 |
| 「購買」「價格」 | 发送商品目录链接 |
| 「教學」 | 发送入门教程合集 |
| 「客服」「問題」 | 转接 FAQ 页面 |

### 好友增长策略
1. Vocus 文章嵌入 LINE 好友链接
2. Dcard 个人简介添加 LINE 账号
3. 举办「加入 LINE 好友送免费工具包」活动
