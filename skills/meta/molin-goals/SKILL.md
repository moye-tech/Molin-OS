## 目标级联规则

每当我派发任务时，任务上下文必须包含：
```
🎯 使命：月收入 ¥30,000
📋 目标：[O1/O2/O3] - [目标描述]
📌 任务：[具体任务]
✅ 成功标准：[可验证的结果]
```

示例：
```
🎯 使命：月收入 ¥30,000
📋 目标：O1 - 闲鱼服务实现稳定月入
📌 任务：为"商业计划书代写"商品撰写详情页文案
✅ 成功标准：文案包含案例展示、定价说明、交付流程
```
