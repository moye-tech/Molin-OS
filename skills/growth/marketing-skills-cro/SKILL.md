# Marketing — CRO (转化率优化)

## Overview

Conversion Rate Optimization framework for digital products and services. Optimize every step of your sales funnel from impression to purchase.

## CRO Workflow

### 1. Funnel Audit
- Map the complete user journey
- Identify drop-off points at each stage
- Quantify: impressions → clicks → engagement → purchase
- Find the biggest leak (highest impact fix)

### 2. Hypothesis Generation
For each drop-off point, generate testable hypotheses:
- "If we change X to Y, conversion will improve by Z%"
- Prioritize by: impact × confidence × ease

### 3. Copy Optimization
- Headline testing: emotional vs functional
- CTA testing: action verbs vs benefit-focused
- Social proof placement
- Urgency/scarcity triggers (ethical)

### 4. Design Optimization
- Above-the-fold content
- Visual hierarchy
- Trust signals (reviews, badges, guarantees)
- Mobile responsiveness

### 5. Measure & Iterate
- Define primary metric (conversion rate)
- Track secondary metrics (bounce rate, time on page)
- Statistical significance check
- Winner implementation → next test

## Xianyu-Specific CRO

| Element | Optimization |
|---------|-------------|
| Title | Include keyword + benefit + price signal |
| Cover image | Before/after split, text overlay |
| Description | Problem → solution → social proof → CTA |
| Pricing display | Show value first, price last |
| Response time | Auto-reply within 10 seconds |

## Quick Audit Checklist
- [ ] Does the title make the benefit clear in 3 seconds?
- [ ] Is there social proof (ratings, reviews, case studies)?
- [ ] Is the CTA specific and action-oriented?
- [ ] Are there trust signals (guarantee, credentials)?
- [ ] Is the price justified with value comparison?
