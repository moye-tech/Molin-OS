# Marketing — Copywriting (文案优化)

## Persuasion Frameworks

### AIDA
```
Attention → Interest → Desire → Action
```

### PAS (Problem-Agitate-Solve)
```
Problem: "AI绘画太贵了？"
Agitate: "市面上一张要200-500，还要等3天"
Solve: "我用ComfyUI工作流，¥50一张，2小时出图"
```

### 4P (Promise-Picture-Proof-Push)
```
Promise: "让你的简历面试率提升3倍"
Picture: "想象一下，投10家收到8个面试邀请"
Proof: "已帮助200+求职者拿到大厂offer，平均面试率从12%提升到38%"
Push: "现在下单，24小时内出稿，不满意全额退款"
```

## Headline Patterns for Xianyu

| Pattern | Example |
|---------|---------|
| Number + Result | "3天出稿，已帮50+创业者拿到融资" |
| Pain + Solution | "PPT做了3天还被老板骂？我用AI30分钟搞定" |
| Social Proof | "500强PM都在用的PRD模板" |
| Urgency | "本月仅接10单，已约7单" |
| Comparison | "比去猪八戒省80%的钱，质量一样" |

## Emotional Triggers

| Trigger | When to Use |
|---------|------------|
| Fear of missing out | Limited slots, time-limited offers |
| Social proof | Customer reviews, case numbers |
| Authority | Certifications, client logos |
| Reciprocity | Free sample, free consultation first |
| Scarcity | "本月最后3个名额" |
| Specificity | "23位客户的真实反馈" vs "很多客户" |

## Platform Voice Guide

| Platform | Tone | Length |
|----------|------|--------|
| 闲鱼 | 亲切+专业 | 100-300字 |
| 小红书 | 真实+有用 | 500-1500字 |
| 猪八戒 | 专业+可信 | 500-2000字 |
| 微信 | 深度+价值 | 800-3000字 |
