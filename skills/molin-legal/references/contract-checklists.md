# 合同类型检查清单

## SaaS / 软件即服务
- [ ] 服务级别协议(SLA) — 可用性(≥99.9%)、响应时间、退款政策
- [ ] 数据处理协议(DPA) — GDPR第28条合规
- [ ] 使用限制 — 用户数/API调用/存储限制
- [ ] 数据出口/迁移 — 合同终止时数据获取
- [ ] 知识产权 — 用户数据归属、反馈使用权
- [ ] 费用与付款 — 订阅费、超额费、涨价通知期
- [ ] 终止 — 提前终止权、SaaS友好终止条款
- [ ] 安全 — SOC2/ISO27001认证、事件通知

## 雇佣/劳动合同
- [ ] 竞业限制 — 范围(时间/地域/业务)合理性
- [ ] 知识产权归属 — 职务发明归属、员工作品
- [ ] 保密义务 — 范围/期限/例外
- [ ] 薪酬与福利 — 基本/奖金/股权/期权/福利
- [ ] 终止 — 通知期、离职补偿、善意终止
- [ ] 非招揽 — 前后员工的限制
- [ ] 可分割性 — 部分无效不影响整体

## MSA (主服务协议)
- [ ] 服务范围(SOW机制) — 如何定义具体工作
- [ ] 付款条件 — 里程碑/工时/固定价格
- [ ] 变更管理 — 范围变更流程
- [ ] 验收标准 — 交付物验收流程
- [ ] 知识产权 — 工作产品归属、背景IP
- [ ] 保密 — 双向/单向
- [ ] 终止 — 有因/无因/部分终止
- [ ] 责任上限 — 通常为费用总额(M)或1M(M)

## NDA (保密协议)
- [ ] 保密信息定义 — 书面标记/口头确认/合理人标准
- [ ] 保密期限 — 常见2-5年
- [ ] 允许披露 — 员工/顾问/法律要求
- [ ] 例外 — 公共/独立开发/第三方来源
- [ ] 返还/销毁 — 合同终止后
- [ ] 管辖法律

## 合伙/合资协议
- [ ] 管理结构 — 决策权、投票权
- [ ] 利润/损失分配 — 比例、分步机制
- [ ] 出资 — 现金/实物/IP/劳力折股
- [ ] 退出 — 出售/买断/IPO/解散
- [ ] 竞业 — 合伙期间及退出后
- [ ] 知识产权 — 新IP归属、背景IP许可
- [ ] 僵局解决 — 分家/俄式轮盘/拍卖

## 租赁合同
- [ ] 租期 — 起始/到期/续租权
- [ ] 租金 — 基础/物业/停车/递增机制
- [ ] 押金 — 金额/返还条件
- [ ] 维修责任 — 结构性/日常/设备
- [ ] 转租/分租 — 是否允许
- [ ] 提前终止 — 罚则/通知期
- [ ] 用途限制 — 允许的经营活动

## 供应商/采购合同
- [ ] 货物规格 — 质量标准、检验权
- [ ] 交付 — 时间、地点、运输、保险
- [ ] 价格 — 单位/折扣/涨价机制
- [ ] 付款 — 账期/方式/L/C
- [ ] 质保 — 期限、范围、补救措施
- [ ] 责任 — 产品责任、召回
- [ ] 知识产权 — 侵权赔偿

## 独立承包商/自由职业
- [ ] 工作范围 — 明确定义交付物
- [ ] 交付时间表 — 里程碑+截止日期
- [ ] 报酬 — 金额/付款触发/账期
- [ ] 知识产权 — 成果归属(保留背景IP)
- [ ] 作品集权限 — 署名权/使用权
- [ ] 修订限制 — 免费修订次数/额外费用
- [ ] 终止 — 通知期/杀青费/已完成工作付款
- [ ] 分类合规 — IRS 20因素/ABC测试

## 许可协议
- [ ] 许可范围 — 地域/期限/用途/用户数/设备数
- [ ] 许可类型 — 独占/排他/普通/分许可权
- [ ] 使用限制 — 反向工程/修改/竞品使用
- [ ] 许可费 — 预付/版税/最低保证金
- [ ] 审计权 — 使用量核查
- [ ] 终止 — 违约/破产/便利终止
- [ ] 升级/维护 — 新版本/补丁/支持

## 工作说明书(SOW)
- [ ] 工作描述 — 明确、可验证交付物
- [ ] 时间线 — 里程碑日期
- [ ] 交付标准 — 验收标准
- [ ] 团队 — 关键人员/替代机制
- [ ] 预算 — 费用明细/限额/超预算审批
- [ ] 变更流程 — SOW修订机制
- [ ] 签署 — 双方授权代表签名
