# TradingAgents — 多视角金融分析

## Overview

Apply the TradingAgents multi-agent analytical framework to any financial question. Instead of a single analysis, deploy 4 specialized analytical perspectives simultaneously, then synthesize into a comprehensive assessment.

## The Four Analyst Perspectives

### 1. Fundamental Analyst
- Revenue, profit margins, cash flow
- Market position, competitive moat
- Growth trajectory, unit economics
- Management quality, corporate governance

### 2. Sentiment Analyst
- News sentiment, social media buzz
- Analyst ratings changes
- Insider trading patterns
- Market narrative (bull/bear case)

### 3. Technical Analyst
- Price trends, support/resistance
- Volume analysis, momentum indicators
- Pattern recognition
- Market structure

### 4. Risk Manager
- Downside scenarios, black swan events
- Correlation risks, concentration risks
- Regulatory/geopolitical risks
- Position sizing, stop-loss strategy

## When to Use

- User asks about stocks, crypto, or any investment
- User wants to evaluate a business opportunity
- User needs multi-angle market analysis
- User asks "is this a good investment?"

## Analysis Protocol

For any financial question, provide analysis through ALL four perspectives, then synthesize:

```
FUNDAMENTAL: [Key metrics, competitive position, growth outlook]
SENTIMENT: [Market mood, news narrative, social signals]
TECHNICAL: [Price action, volume, key levels]
RISK: [Major risks, worst-case scenarios, risk mitigation]
─────────────────────────────────────────────
SYNTHESIS: [Weighted conclusion with confidence level]
```

## Limitations

- Not real-time market data (use web search for current prices)
- Analysis is framework-based, not predictive
- Always include disclaimer: not financial advice
