# zhubajie-automation · 猪八戒自动化
# 归属: Agent C (side) 独占 · 版本: v1.0

## 技能身份
猪八戒平台的自动化接单和消息管理。
功能类似 xianyu-automation，适配猪八戒平台特性。

## 触发词
触发词: 猪八戒|八戒|外包|接单|威客
触发词: 猪八戒消息|任务大厅

## 消息分级处理规则
L0自动回复（常见询价）:
  多少钱|能做吗|有案例吗|怎么合作
  → 3分钟内自动发送首回复模板

L1提醒（成交意向）:
  有预算|什么时候能做|打算购买
  → 发飞书T0通知墨烨

L2审批（报价>¥1000）:
  → 发飞书T2审批卡，等确认后发报价

## 首回复模板
「您好！感谢咨询~ 我专注AI工具定制服务，
可承接Agent开发、RAG知识库、自动化流程等项目。
请问您具体想实现什么功能？详细描述一下需求 😊」

## 平台差异
- 猪八戒用户更专业，话术可偏技术
- 竞标模式需要写投标方案（比闲鱼更正式）
- 平台抽成约20%，报价时需含平台费用

## 当前经验规则（动态更新）
- 猪八戒成交价通常比闲鱼低15-20%（竞争更激烈）
- 含技术方案的投标中标率高2倍
