# 2026 小红书双引擎 SEO 参考

## 平台算法重大变更

2026年小红书已完成全面算法重构：
- **从**：流量赛马（推荐单引擎）
- **到**：搜索 + 推荐双引擎（价值深耕）
- **含义**：笔记在推荐流量中表现好 → 搜索流量同步增加。用户在搜索某关键词后的互动行为 → 影响相关推荐的权重。搜索排名会直接影响推荐流量的持续性。

## 算法关键词抓取位置（5个位置）

| 位置 | 权重 | 说明 |
|:-----|:-----|:-----|
| 标题 | 最高 | 核心搜索词必须出现在标题 |
| 首段文案 | 高 | 算法用首段做内容分类，决定推荐给谁 |
| 正文内容 | 中 | 关键词自然分布，密度1-3% |
| 话题标签 | 中高 | 最多10个，标签=搜索入口 |
| 评论区高频词 | 低 | 互动后生效，间接影响 |

## 小红书 SEO 优化清单

- [ ] 标题含2-3个核心搜索关键词（避免纯叙事/情绪标题）
- [ ] 首段（前40-60字）含核心关键词 + 结论先行
- [ ] 正文关键词自然分布，不堆砌
- [ ] 标签满10个（平台上限），含长尾标签
- [ ] 封面文字含补充关键词
- [ ] 评论区引导互动（高频词会被抓取）
