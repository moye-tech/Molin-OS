## 四、系统前后对比

| 维度 | 实施前 | 实施后 |
|------|-------|-------|
| Agent 路由方式 | 靠 LLM 随机判断用什么工具 | **查 SOUL.md 触发表 → 确定调用技能** |
| 跨线协作 | 手动 delegate_task 硬编码 | **relay/shared/cross_request.json 队列** |
| 热门技能覆盖 | 部分有 SKILL.md，部分无 | **9个新技能补齐，触发词=执行路径** |
| 联网调研能力 | 无（靠训练数据） | **gpt-researcher 实时联网，64MB+依赖** |
| 可观测性 | 黑箱 | **langfuse 追踪 + @trace_skill 装饰器** |
| 知识时效 | 依赖上次更新的数据 | **3个 Cron 每天自动拉取最新** |
| 记忆桥接 | Supermemory 单点 | **memory-bridge 桥接 Supermemory + Obsidian** |
