# analytics-tracking · 内容效果数据追踪
# 归属: Agent A (media) · 版本: v1.0
## 触发词
触发词: 数据|互动量|点赞|收藏率|阅读量|播放量|涨粉
触发词: 效果分析|爆款分析|内容复盘|哪条最好
## 执行步骤
Step 1: 读取 relay/media/analytics.json（昨日数据）
Step 2: 计算各维度指标（点赞率/收藏率/转化率）
Step 3: 对比本周均值，识别异常（>均值2倍 = 爆款候选）
Step 4: 爆款分析：标题/时段/话题/格式哪个因素决定的
Step 5: 输出T1数据卡片
## 输出格式
{ "top3": [{...}], "bottom3": [{...}], "insight": "本期算法偏好X类内容" }
